**Herramientas IA (cloud):**

**Modelos Open Source para Self-Hosting (2025-2026):**
| Modelo | Empresa | Destacado |
|--------|---------|-----------|
| **DeepSeek R1/V3** | DeepSeek | 671B params, MoE, MIT license, líder en razonamiento |
| **Qwen3-235B** | Alibaba | 235B params, supera GPT-4o en benchmarks |
| **Kimi K2** | Moonshot AI | Top modelo open-weight mundial |
| **GLM-4.5** | Zhipu AI | 355B params, multimodal |

**Ventajas Self-Hosted:**
- Privacidad total de datos (no salen del servidor)
- Sin costes por token / API
- Cumplimiento normativo (GDPR, datos sensibles)
- Personalización y fine-tuning sin restricciones
- Independencia de proveedores cloud
